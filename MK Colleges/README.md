# MK Institution Website

## 📌 Project Description

MK Institution Website is a modern, responsive, and interactive educational institution landing page built using HTML, CSS, and JavaScript. This website is designed for showcasing MK Institution’s academic programmes, colleges, facilities, admissions, campus gallery, and contact details in a professional and user-friendly layout.

The website represents an educational institution established in 1978, located in Chennai, Tamil Nadu. It includes a clean navigation bar, hero section with video background, statistics counters, course filtering, admission fee table, image gallery with lightbox preview, application modal form, and contact form validation.

## ✨ Features

- Modern landing page design for an educational institution
- Fixed responsive navigation bar
- Hero section with video background
- Student, course, placement, and excellence statistics
- About section with institution highlights
- Course cards with department filter options
- Colleges section for Engineering, Arts & Science, and Medical campuses
- Facilities section with campus life details
- Admission fee table with seat availability
- Apply Now modal form
- Campus gallery with lightbox image preview
- Contact section with form validation
- Footer with quick links and social media links
- Back-to-top button
- Mobile responsive layout

## 🎯 Purpose of the Project

The main purpose of this project is to create a professional website for an educational institution where students can explore courses, view admission details, check facilities, see campus images, and contact the institution easily.

## 🚀 How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Run the `index.html` file in any web browser.
4. Make sure all images and videos are placed in the correct path.
